# Customer Churn and Customer Spending Prediction in Telecommunications Using Machine Learning

## How to Run the Scripts

We worked in notebook-style IDEs (Jupyter Notebook and Google Colab), so our scripts can be opened directly in Google Colab without the need to set up a virtual environment. We also read the raw data from our public GitHub repository, so we do not have problems with file paths and our scripts can be executed from any computer. Some of our notebooks are hosted on GitHub and include a button to open them in Colab, while others are opened directly in Google Colab. In both cases, the notebooks can be accessed and executed successfully.
