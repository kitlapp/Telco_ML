How to Run the Scripts
We worked in notebook style IDE's (Jupyter) and Google Colab directly so that they our scripts can be all opened in Google Colab without the need to set up a virtual environment. We also read the data from our public GitHub repository so that we won't have any problem with the paths and our scripts could be executed from every computer.
